# [WIP] YourTimer

## Run

ピンクノイズを生成
```bash
cd _scripts
uv sync
uv run main.py
```

お好きな mp3 ファイルを拾ってきて、timeup.mp3 にリネームしてください(終了時の音として使用します)。

```bash
go run main.go
```
